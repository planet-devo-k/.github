<summary><b>Explore the Planet</b></summary>
<br/>
Planet Devo K는 서로의 지식과 경험을 레버리지 삼아 건강한 성장 궤도를 함께 만들어가는 빌더들의 공동체입니다.
<br/>혼자 공부할 때 1의 시간은 1의 결과에 머물지만, 공유되는 순간 그 노력은 레버리지가 되어 n 이상의 가치로 확장됩니다.
<br/>지식을 나누는 행위가 소모가 아닌 '증폭'이 되는 이 특별한 경험을 함께하고 싶습니다.
<br/>
  
```js
const PlanetDevoK = {
  mission: "Amplifica",
  growing: true,
  
  leverage: (effort, members) => {
    const n = members.length;
    return effort * Math.pow(n, 2); 
  },
 
async startGrowing(knowledge) {
    while (this.growing) {
      await share(knowledge);      
      const insights = receive(); 

      if (this.isStuck) {
        ask_community();   
      }
    }
  }
};
```

## Activities
- **학습 몰입**: 기술적 원리 탐구와 심도 있는 학습 
- **집단 지성 공유** : 인사이트, 개인의 학습 경험 및 유용한 정보 공유
- **모멘텀 형성**: 학습의 관성을 만들고 끊김 없는 성장 흐름을 유지
- **네트워킹**: 비슷한 고민을 하는 동료와의 네트워킹
- **협업 역량 확장**: 기술적 커뮤니케이션 능력 확장


## Ground Rules
우리의 여정이 지속 가능한 보상이 되기 위해 다음의 원칙을 선언합니다.<br/>
<br/>:one: **경쟁보다 협업**: 서로의 생각과 관점을 존중하고 배웁니다. 어떤 의견도 편하게 나눌 수 있는 환경을 함께 만들어요.
<br/>:two: **반응과 피드백**: 침묵은 오해를, 반응은 신뢰를 만듭니다.
<br/>:three: **책임감 있는 참여**: 매주 정한 학습 분량은 스스로 책임집니다. 어려울 땐 미리 소통해요.
<br/>:four: **아낌없이 공유하기**: 아는 것은 나누고, 모르는 것은 질문합니다. **공유는 팀의 자산, 질문은 모두의 학습 기회가 됩니다.**
<br/>:five: **질문은 자유롭게, 설명은 친절하게**: 모르는 건 창피한 게 아니에요. 
<br/>:six: **타인의 작업을 평가하기보다 그 안의 고민을 먼저 들여다봅니다.**
<br/>:seven: **PR은 스스로의 정리이고, 리뷰는 서로의 배움입니다.** 
<br/>:eight: **지속 가능한 커뮤니티를 위해 같이 가꿔나갑니다.** 


## Manifesto 
서로의 사고를 공유하는 것이 가장 빠른 성장의 지름길이라는 믿음<br/>
<br/>:one:**엔진: 흔들리는 의지를 잡아주는 약속**
<br/>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;혼자라면 쉽게 멈췄을 순간에도, '함께'라는 약속이 우리를 계속 움직이게 합니다.
<br/>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;우리는 서로의 페이스메이커가 되어 완주를 돕고, 끝까지 함께하는 힘을 믿습니다.
<br/>:two:**궤도: 지속 가능한 모멘텀**
<br/>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;우리는 꾸준함이 만드는 비선형적 성장을 증명합니다.
<br/>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;우리는 학습의 관성을 만들고 끊김 없는 성장 흐름을 유지합니다.
<br/>:three:**엠플리피카: 1의 노력으로 N의 성장을, 지식 레버리지**
<br/>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;공유는 소모가 아닌 복리적 증폭입니다.
<br/>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;우리는 서로의 관점을 나누며 지식을 교차 검증하고 사고의 지평을 넓힙니다.
<br/>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;우리는 개인의 1시간을, 팀 전체의 가치로 확장하는 지식의 복리를 추구합니다.
